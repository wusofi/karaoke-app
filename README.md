# Karaoke Web App

Fitur:
- Pencarian lagu dari YouTube
- Tambah ke antrean + drag & drop
- Mode TV (dual screen)
- Tema gelap/terang
- Favorit lagu disimpan
- Auto-play berikutnya

Cara pakai:
1. Buka `index.html`
2. Klik tombol TV Mode
3. Tambah lagu dari pencarian atau link
4. Nikmati karaoke 🎤
